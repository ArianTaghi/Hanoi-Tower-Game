# 🎮Hanoi-Tower-Game
A hanoi tower-puzzel game with auto solve system.

In this java game:

- You can play game by changing the position of cylinders
- there is 3 rays which can contain cylinders
- You win when you move all cylinders from one ray to anonther
- This game also has an auto solve mechanism which solves the puzzle using recursive methods technique

  ---
Requirements to run:
  - java 25+
  - javafx 25.0.2+

made with intelliJ idea
